# C# OOP Concepts - Learning Guide

A collection of C# code examples demonstrating core Object-Oriented Programming (OOP) concepts.

## Topics Covered

| File | Concept | Description |
|------|---------|-------------|
| `ClassesandObjects.cs` | Classes & Objects | Creating classes and instantiating objects |
| `ClassMembers.cs` | Class Members | Fields and multiple object instances |
| `Constructors.cs` | Constructors | Parameterized constructors for initialization |
| `AccessModifiers.cs` | Access Modifiers | `private`, `public` access control |
| `PropertiesandEncapsulation.cs` | Properties & Encapsulation | Getters/Setters for data protection |
| `Inheritance.cs` | Inheritance | Parent-child class relationships |
| `Polymorphism.cs` | Polymorphism | Method overriding with `virtual`/`override` |
| `Abstraction.cs` | Abstraction | Abstract classes and methods |
| `Interface.cs` | Interfaces | Multiple interface implementation |
| `enum.cs` | Enumerations | Named constants with `enum` |
| `files.cs` | File I/O | Reading and writing files |
| `prg.cs` | Exception Handling | `try-catch-finally` blocks |

## Requirements

- .NET 8.0 SDK or later

## How to Run

```bash
# Build the project
dotnet build

# Run the project
dotnet run
```

## OOP Pillars Demonstrated

1. **Encapsulation** - Hiding internal data using private fields and public properties
2. **Inheritance** - Deriving child classes from parent classes
3. **Polymorphism** - Same method behaving differently in derived classes
4. **Abstraction** - Hiding complexity using abstract classes and interfaces
